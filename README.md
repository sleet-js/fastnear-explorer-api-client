# fastnear-explorer-api-client

To install dependencies:

```bash
bun install
bun update --latest
bun run tsc --noEmit
# bun run index.ts
bun run build
bun publish --dry-run
bunx npm login
bun publish --access public
```

This project was created using `bun init` in bun v1.3.11. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
